# 🏙️ City Chat — Real-Time Chat System

## 🔴 Live Demo → [Click Here to Open](https://chat-city-ejnb9xyz4-venkycodexs-projects.vercel.app)

> A real-time city-based chat system built with React, Node.js, Express and Socket.IO.

---

## How to Use

**As a User:**
1. Enter your name
2. Select a city (London, Birmingham, Manchester)
3. Start chatting in real-time

**As an Admin:**
1. Click "Admin? Login here" on the login screen
2. Enter your name and select a city to monitor
3. Reply to users instantly

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js |
| Backend | Node.js + Express |
| Real-time | Socket.IO |
| Frontend Host | Vercel |
| Backend Host | Render |

---

## Run Locally

```bash
git clone https://github.com/VenkyCodeX/chat_city.git
cd chat_city

# Backend
cd server
npm install
npm start

# Frontend (new terminal)
cd ../client
npm install
npm start
```
